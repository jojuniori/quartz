# 10 款最新发现的免费小工具

![[知识/软件/_assets/10-free-tools-2026-05-20/cover-1772530347899-1.jpg]]

## 一句话总结

这期是偏“开发者 / AI / 本地工具”的小工具合集，亮点集中在录屏编辑、旧安卓复用、GitHub 文件夹下载、AI Skill 生成、macOS 虚拟机隔离、本地照片管理和 AI 音频处理。

## Recordly：开源屏幕录制器和编辑器

跨平台录屏工具。重点不是“能录”，而是录完之后能做出更像产品演示的视频：自动强调重点区域、平滑光标、添加壁纸/纯色/渐变/模糊/留白/阴影边框，也支持缩放、裁剪、变速区域、注释和额外音频区域。

![[知识/软件/_assets/10-free-tools-2026-05-20/recordly-demo.mp4]]

## linux-android：重新利用旧安卓手机

把旧安卓手机变成 Linux 桌面或 Home Assistant 服务器的项目。无需电脑、无需 root、无需云，核心依赖 Termux。大致要求是 3GB+ 内存、10GB+ 空间，最好是高通 Snapdragon 芯片。

![[知识/软件/_assets/10-free-tools-2026-05-20/linux-android.avif]]

可以配 XFCE4 桌面、GPU 加速、SSH、SCP 等；作为低成本家庭服务器、轻量 Linux 环境或 AI Agent 运行环境都值得研究。

## ghgrab：从 GitHub 子文件夹下载文件

GitHub 页面不能直接下载某一个子文件夹，经常只能 clone 或下载整个仓库。ghgrab 是命令行工具，可以在不 clone 的情况下下载指定文件或文件夹，适合只取模板、示例目录或资源目录。

![[知识/软件/_assets/10-free-tools-2026-05-20/ghgrab.avif]]

## Waza：生成 Skill 的工具

当有重复性工作想交给 AI，但还不知道怎么写成 Skill 时，可以先用 Waza 和 AI 对话。它会追问使用场景、目标、流程细节等内容，最后生成可重复使用的 Skill。

适合把“经常要让 AI 做的一串操作”沉淀成稳定流程。

## GhostVM：macOS 虚拟机隔离环境

GhostVM 是 Apple Silicon 上的 macOS 虚拟机管理工具，使用 Apple 的 Virtualization.framework。每个虚拟机都是独立 `.GhostVM` 包，可以复制、移动、备份。

![[知识/软件/_assets/10-free-tools-2026-05-20/ghostvm-hero.avif]]

适合软件测试、临时环境、隔离运行未知程序，也适合给 AI Agent 准备互不影响的小环境。

## SoloMD：整合 AI 的 Markdown 编辑器

Tauri 2 + Rust + WebView 的 Markdown 编辑器，主打 AI 工作流：内置 Agent 面板、定时 Agent Recipe、AutoGit 分支沙箱、可重放 Agent Trace、多 vault MCP Federation、本地语义搜索等。

可以看作是偏 AI-native 的 Markdown 编辑器，而不是单纯的 Typora 替代品。

## PeekDesktop：让 Windows 桌面像 macOS 一样

Windows 工具：点击桌面任意一处，其他窗口自动移开，露出主桌面；再点一下桌面，窗口恢复。效果接近 macOS 的“点桌面显示桌面”。

![[知识/软件/_assets/10-free-tools-2026-05-20/peekdesktop-demo.mp4]]

## Lap：免费开源本地照片管理器

开源、本地优先的照片管理器。它不强制上传、不强制导入，以本地文件夹为中心，目标是流畅管理大照片库。

![[知识/软件/_assets/10-free-tools-2026-05-20/lap-demo.mp4]]

主要能力包括：

- 处理 10 万个以上文件的照片库
- 直接使用本地文件夹
- 简洁现代的桌面界面
- 本地 AI 搜索和测试中的人脸识别

## Guizang PPT Skill：网页 PPT / 配图 / 封面

面向 Claude Code / Codex 等 Agent 环境的 PPT 技能，用于生成单文件 HTML 横向翻页 PPT、PPT 配图和多平台封面。

这个方向很适合沉淀成自己的“内容生成流水线”：提纲、配图、封面、演示页都让 Agent 一次性生成，再人工修订。

## voice-pro：AI 语音工具集合

把多种 AI 音频能力塞进一个本地网页界面里，包括视频/音频转文字、语音翻译、AI 配音、TTS、声音克隆、人声分离、YouTube 下载与提取音频、字幕生成、多语言配音等。

![[知识/软件/_assets/10-free-tools-2026-05-20/voice-pro-main.avif]]

可以理解为本地版 AI 配音 + AI 字幕 + AI 翻译工作室。需要注意：项目已经停滞开发，适合参考能力组合，实际使用前要确认依赖和安全性。

## 我可能会优先试的

- [ ] **ghgrab**：很适合下载 GitHub 项目的单个模板、示例目录或资源目录。
- [ ] **GhostVM**：如果要跑风险较高的软件或给 Agent 建隔离环境，值得研究。
- [ ] **Waza**：和当前 Codex / Skill 工作流关系很近，可以看看生成的 Skill 质量。
- [ ] **Lap**：如果照片库越来越大，可以作为 Picasa / FastStone 替代品观察。
- [ ] **Recordly**：适合录制工具演示，可能比普通录屏更省后期。
