## 维生素 C

### 特性

- **经典水溶性抗氧化剂**：人体内最重要的水溶性抗氧化维生素，需要每日补充，无法在体内长期储存。
- **剂量依赖性吸收**：小剂量时吸收率高（30-60mg 时接近 100%），大剂量时吸收率显著下降（1500mg 时仅 49%，3000mg 时仅 36%），展现出智能化吸收特征。
- **多样化补充剂形式**：
  - **普通抗坏血酸**：最常见形式，酸性较强，空腹可能引起胃部不适
  - **缓释型维生素 C**：延长释放时间，维持更稳定的血药浓度，减少肾脏排泄损失
  - **天然来源（如针叶樱桃）**：含有维生素 C 复合物和生物类黄酮，可能具有更好的生物利用度
  - **缓冲型（如抗坏血酸钙）**：pH 中性，对胃肠道更温和
- **协同营养网络**：与维生素 E 形成抗氧化循环，能够再生维生素 E；与生物类黄酮协同增强抗氧化效果。

### 相关影响

- **胶原蛋白合成基石**：作为羟化酶的必需辅因子，参与胶原蛋白中脯氨酸和赖氨酸的羟化反应，是维持皮肤弹性、血管完整性、骨骼健康的生化基础。
- **免疫系统全面支持**：
  - 增强中性粒细胞的趋化性和吞噬能力
  - 促进淋巴细胞增殖和抗体生成
  - 支持皮肤和黏膜屏障功能
  - 在感染期间需求量可增加 10-20 倍
- **强效抗氧化防护**：清除水溶性自由基（羟基自由基、超氧阴离子等），保护细胞膜和 DNA 免受氧化损伤，是细胞内外抗氧化防御的第一道防线。
- **矿物质吸收促进器**：将三价铁还原为易吸收的二价铁，显著提高植物性食物中非血红素铁的生物利用度，对素食者和缺铁性贫血患者尤为重要。
- **神经递质合成参与**：参与多巴胺转化为去甲肾上腺素的过程，影响情绪调节和认知功能。
- **心血管内皮保护**：维护血管内皮细胞完整性，参与一氧化氮合成，有助于血管舒张和血压调节。

### 缺乏影响

- **坏血病（重度缺乏）**：胶原蛋白合成严重受损，表现为牙龈出血肿胀、皮下出血点、伤口愈合缓慢、关节疼痛，严重时可致命。
- **免疫功能受损**：反复感冒感染、伤口愈合慢、皮肤黏膜屏障功能下降，感染恢复期延长。
- **疲劳和贫血**：铁吸收不良导致的缺铁性贫血，表现为疲劳乏力、注意力不集中、运动耐量下降。
- **皮肤老化加速**：胶原蛋白合成不足导致皮肤弹性下降、皱纹增加、伤口愈合能力减弱。
- **血管脆性增加**：毛细血管壁脆弱，容易出现瘀斑和出血倾向。

### 自然补充

- **超高含量水果**：针叶樱桃（1300-3000mg/100g）、卡卡杜李子（2000-5000mg/100g，世界最高）、猕猴桃（90mg/100g）
- **柑橘类经典来源**：橙子、柠檬、柚子、橘子等，维生素 C 含量适中但稳定易得
- **蔬菜类丰富来源**：
  - **十字花科**：西兰花、花椰菜、球芽甘蓝
  - **茄科**：红甜椒（维生素 C 含量超过橙子）、番茄
  - **绿叶蔬菜**：菠菜、芥蓝、羽衣甘蓝
- **保存和烹饪注意事项**：
  - 维生素 C 对热、光、氧气敏感，长时间加热会大量流失
  - 生食或短时间蒸煮能最大程度保留含量
  - 切开后应尽快食用，避免长时间暴露在空气中
  - 冷冻蔬果的维生素 C 含量可能高于长期储存的新鲜产品
